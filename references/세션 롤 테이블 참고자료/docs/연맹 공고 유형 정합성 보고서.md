# 연맹 공고 유형 정합성 보고서

작성일: 2026-07-11

## 기준과 범위

- **신규 기준 데이터:** `tables/league-notice-token-mapping.tsv`의 85개 연맹 공고 유형
- **기존 서술 기준:** `자유 탐사 연맹 공고 유형.md` 및 같은 폴더의 설정 문서
- **범위:** 자유 탐사 연맹 공고만 다룬다. 민간 의뢰는 이 분류에 포함하지 않는다.

신규 표는 세션 롤 테이블에서 사용할 분류·의뢰자 계열·사유 토큰·몬스터 후보의 기준이다. 기존 문서의 상세 서술은 폐기하지 않으며, 아래 대응표를 따라 최신 공고 유형의 배경 설명 또는 후속 전개로 사용한다.

## 공통 제약

- 연맹 공고는 사적 고용 계약이 아니라 미지와 위험을 기록·관측·회수·구조·봉인·검증·제압하는 탐사 과제다.
- 던전 내부 몬스터와 던전 가디언은 던전 경계 밖으로 나오지 않는다.
- 앙그라권 레이드·사냥과 토템 가디언 레이드는 단독 사냥이 아니라 원정대 단위 임무다.
- `기밀`은 의뢰 종류가 아닌 보안 등급이다.
- 기밀 `재앙 대응` 공고의 주요 배후는 앙그라의 안개를 섬기는 흑암교단, **공포의 의회**다. 다만 조사 초기에는 조직의 존재를 단정하지 않고 잔향·기록 조작·금기 의식·오염 사건으로 제한 공개한다.

## 기존 유형과 신규 유형의 대응

### 직접 유지 또는 명칭 정리

- 기존 `민생·구조`, `사냥꾼`, `던전`, `잔향·침식`, `잔향의 룬`, `정제·자원`, `기록·검증`, `시설·방어·봉인`, `분쟁·중립`, `앙그라권 탐사`, `앙그라권 사냥꾼 배속`의 핵심 설명은 유지한다.
- 새 표의 공고 그룹명으로 문서 목차를 정렬한다. `사냥꾼 공고`는 몬스터 사냥·잔향 변이 사냥·던전 사냥·앙그라권 레이드·사냥의 상위 설명으로 둔다.
- `특수 공고`는 표시용 의뢰 종류가 아니라 `기밀 / 재앙 대응`으로 정리한다.

## 일치성 검증 규칙

새 연맹 공고를 추가하거나 수정할 때 다음을 함께 검토한다.

1. TSV에 `공고 그룹`, `의뢰 유형`, `보안 등급`, `의뢰인 계열`, `사유 토큰 세트`, `몬스터 후보`가 있는가.
2. `자유 탐사 연맹 공고 유형.md` 또는 이 보고서에 해당 유형의 목적과 재분류 조건이 있는가.
3. 앙그라의 안개, 앙그라의 잔향, 검은 광기, 던전, 공포의 의회 문서와 충돌하지 않는가.
4. 던전 경계, 토템 가디언 레이드, 공포의 의회 기밀 취급 규칙을 위반하지 않는가.
5. 기존 공고를 대체할 경우 삭제가 아니라 대응·흡수·재배치 상태가 기록되어 있는가.
