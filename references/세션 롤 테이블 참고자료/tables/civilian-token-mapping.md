# 민간 의뢰 분류별 의뢰자/의뢰 사유 토큰 매핑

민간 의뢰는 공식 공고 유형을 따르지 않는다. 의뢰 사유는 단어 토큰 배열로 처리한다.

- 의뢰 대분류: 민간 의뢰
- 의뢰 종류: 생활 피해

- 등장 예시 몬스터: 점액체 몬스터 / 야수 몬스터 / 괴수 몬스터
- 의뢰 사유 토큰 세트:
  - [창고, 식량, 감소]
  - [우물, 오염]
