# 세션 생성 결과 UI 예시

## 예시 1: 연맹 의뢰 / 개척자

```txt
의뢰 대분류: 연맹 의뢰
의뢰 종류: 개척자
의뢰인: 경계 관측관
의뢰 사유: [관측탑, 신호, 두절]
테마: 안개 추적자의 테마
등장 예시 몬스터: 변이체 몬스터
```
