# 몬스터 롤 테이블 경량 데이터

이 데이터셋은 완성된 몬스터 생태 보고서를 만드는 용도가 아니라,
아이디어가 없을 때 버튼을 눌러 몬스터 콘셉트를 빠르게 얻는 용도로 정리했습니다.

## 남긴 7개 개념

1. 출현 맥락
2. 몬스터 대분류
3. 몬스터 세부 분류
4. 크기
5. 집단 구조
6. 행동
7. 전투 방식

## 데이터 규모

- CSV 파일: 7개
- 전체 데이터 행: 435개
- 대분류: 14개
- 세부 분류: 274개
- 집단 구조: 21개
- 전투 방식: 87개

기존 정규화 데이터 11,280행에서 경량판 435행으로 줄였습니다.

## 주요 변경

- `현장 상태`, `이동`, `감각`, `흔적`, `위험 징후`, `드랍`, `자원`을 제거했습니다.
- `우두머리`와 `사회성`을 `집단 구조` 하나로 통합했습니다.
- `공격 방식`과 `피해 유형`을 `전투 방식` 하나로 통합했습니다.
- 무작위 선택에 사용할 수 있도록 모든 CSV에 `weight`를 넣었습니다.
- 세부 분류는 기존 274개를 그대로 유지했습니다.
- 모든 CSV는 UTF-8, BOM 없음, LF 줄바꿈 형식입니다.

## 권장 출력 형식

```text
출현 맥락: 폐허·유적
분류: 악마체 ─ 지배자형 악마체
크기: 중형
집단: 소규모 무리의 우두머리
행동: 추격형, 매복형
전투 방식: 근거리 · 타격
```

대분류와 세부 분류는 별도 데이터이지만 화면에서는 한 줄로 합쳐도 됩니다.
`schema_manifest.json`에 파일별 선택 개수와 필터 규칙을 기록했습니다.
