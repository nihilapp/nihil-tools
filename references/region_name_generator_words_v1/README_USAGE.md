# Region Name Generator Word Pack v1

이 패키지는 판타지 지역명 생성기를 만들기 위한 원천 단어 풀입니다.
파일명은 압축 해제 시 인코딩 문제가 적도록 영문/숫자 중심으로 구성했습니다.

## 구성

- `01_word_pools/nouns.txt`: 명사 525개
- `01_word_pools/adjectives.txt`: 형용사 427개
- `01_word_pools/subjects.txt`: 주어 405개
- `01_word_pools/event_adnominals.txt`: 사건형 관형어 672개
- `03_connector_rules/connector_rules.md`: 연결어별 사용 규칙
- `03_connector_rules/slot_pools/`: 연결어 패턴별 명사 후보 풀
- `04_implementation_notes/`: 구현 메모와 TypeScript 예시
- `05_examples/`: 생성 예시

## 기본 생성 패턴

가장 먼저 아래 네 가지를 기본 패턴으로 사용합니다.

```txt
{명사}의 {지역}
{형용사} {지역}
{주어}이/가 {동사} {지역}
{사건형관형어} {지역}
```

예시는 다음과 같습니다.

```txt
침묵의 늪
찬란한 다리
죽음이 도사리는 숲
불타버린 도시
```

## 연결어 패턴의 핵심 규칙

아래 패턴들은 명사를 아무거나 넣으면 안 됩니다.
패턴이 먼저 정해지고, 그 패턴에 맞는 전용 슬롯 풀에서 명사를 뽑아야 합니다.

```txt
{명사}에 잠긴 {지역}
{명사}에 물든 {지역}
{명사}에 삼켜진 {지역}
{명사} 아래의 {지역}
{명사} 너머의 {지역}
{명사} 사이의 {지역}
{명사} 끝의 {지역}
{명사} 없는 {지역}
{명사}을/를 잃은 {지역}
{명사}을/를 기다리는 {지역}
```

예를 들어 `{명사}에 잠긴 {지역}`은 `안개`, `침묵`, `달빛`, `연기`, `심연`처럼 공간을 덮거나 잠기게 만들 수 있는 명사를 써야 합니다.
그래서 `안개에 잠긴 도시`는 자연스럽지만, `왕관에 잠긴 숲`은 일반 자동 생성 결과로는 부적절합니다.

반대로 `{명사}에 물든 {지역}`은 `피`, `노을`, `재`, `녹`, `저주`, `달빛`처럼 색채나 오염, 빛, 흔적을 남길 수 있는 명사가 적합합니다.
그래서 `피에 물든 전장`, `노을에 물든 해안`, `녹에 물든 성채`가 자연스럽습니다.

## 추천 생성 순서

1. 지역명사 풀에서 지역을 하나 고릅니다.
2. 패턴을 하나 고릅니다.
3. 기본 패턴이면 해당 단어 풀에서 바로 뽑습니다.
4. 연결어 패턴이면 `slot_pools`에서 해당 패턴의 명사만 뽑습니다.
5. `이/가`, `을/를`은 받침 여부로 자동 처리합니다.
6. 최종 결과를 만든 뒤 금칙 조합이나 중복 의미를 한 번 걸러냅니다.

## 조사 처리

한국어 조사는 받침 여부에 따라 갈립니다.

```txt
왕 → 왕이, 왕을
마녀 → 마녀가, 마녀를
빛 → 빛이, 빛을
안개 → 안개가, 안개를
```

구현에서는 마지막 글자의 한글 종성 여부를 확인하면 됩니다.

## 주의할 조합

아래처럼 의미가 너무 당연하거나, 물리적 관계가 약한 조합은 필터링하는 편이 좋습니다.

```txt
물의 호수
나무의 숲
왕관에 잠긴 숲
칼날에 잠긴 평원
왕이 흐르는 산맥
```

예외적으로 초현실, 신화, 고유 설정을 의도한다면 허용할 수 있습니다.
다만 일반 자동 생성에서는 낮은 가중치 또는 수동 승인 대상으로 두는 편이 안전합니다.
